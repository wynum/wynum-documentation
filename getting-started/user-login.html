User Management

A User can login/Sign In on the Demo portal and can either work on his listed projects or create a new custom project.

Login Page

Sign In Tab

Step 1 Open your browser and type the URL of the Demo portal.

Step 2 When you access the the URL, you are prompted to Signin or Signup.

Step 3 Sign In by entering Email address and Password credentials.

Forgot Password

Forgot Password can be clicked whenever user forgets the password. On Clicking Forgot Password, the User will be navigated to "Forgot Password" screen.

On Forgot Password screen, enter Email Address and Click on ‘Send Recovery Email’. A Reset Password link will be sent to the User’s email id.

Home Page

After successful login, the home page is displayed, which contains all the Projects that are currently owned by you. This page also contains the list of Projects that are delegated to you. A User can also add Project, check script failure logs and launch Jupyter Notebook related to each project.

Notifications

User can check the notifications by clicking the icon on the top right menu of the home page. A User get notification about getting access to a particular Project as a collaborator or Observer or Stage Writer.

Profile

User can update the profile information by clicking the icon on the top right menu of the home page. The Edit User Profile page is displayed.

In this page, User can update your profile information, such as Name, Phone Number, Organization and Designation.

Change Password

On the next Tab a User can Change/reset the password. Enter New Password details and click on Submit button. Password will be reset successfully and user will be navigated to Login Screen. On successful login, user will be navigated to the Home page.

Logout

User can Logout by clicking the icon on the top right menu of the home page.
